# Salem World - Coming Soon Page

## Overview
A visually stunning "Coming Soon" landing page for Salem World, a psychological therapy and speech therapy center. The page features modern, futuristic design with purple gradients, smooth animations, and bilingual support (English/Arabic).

## Project Structure
```
├── index.html           # Main HTML structure
├── styles.css           # All styling, gradients, and animations
├── script.js            # Particle effects and language switching
└── attached_assets/     # Logo and media files
    └── 6050720661157645623_1761567933898.jpg
```

## Features Implemented
✅ **Animated floating logo** - Salem World logo floats at center with smooth motion effects
✅ **Purple/white gradient background** - Animated gradient using brand colors (#7C4DFF, #B388FF)
✅ **Floating particle system** - 100 glowing particles with connections creating depth
✅ **Loading animation** - Three bouncing circles beneath main content
✅ **Bilingual support** - Seamless Arabic/English language toggle with smooth transitions
✅ **Responsive design** - Optimized for mobile, tablet, and desktop screens
✅ **Professional typography** - Poppins for English, Cairo for Arabic
✅ **Smooth animations** - Float, pulse, glow, and fade effects throughout

## Color Palette

## Technologies Used
- Pure HTML5
- CSS3 (Animations, Gradients, Flexbox)
- Vanilla JavaScript
- Canvas API for particle effects

## Server
The site runs on a simple Python HTTP server on port 5000.
